* Running a container

```
  docker container run -d --name ngnix ngnix 
```

* List containers

```
   docker container ps
```

* Stop a container

```
   docker stop nginx
```

* Restart a stopped container

```
  docker start nginx
```

* Rmove a stopped container

```
  docker rm ngix
```
